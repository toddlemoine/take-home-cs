# Take Home Exercises

This repos contains files and source related to some take-home
exercises.

## Tables Generator

To see the exercise results, do the following:

```
cd 1_tables
npm install
npm serve
```

To run tests,

```
cd 1_tables
npm test
```

All source code and related output is in the 2_numbers directory.
